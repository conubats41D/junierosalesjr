## Installation

To run this project locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/conubats41d/junierosalesjr.git
   ```
2. Open `index.html` in your browser.

No additional dependencies or build steps are required.

---

## Usage

* Click the **hamburger menu** on mobile devices to expand/collapse navigation.
* The **typewriter effect** dynamically changes the text under your name.
* Navigate through the portfolio to view projects and skills.
